Local Storage

The local storage stores an application data so it does not get lost when the application closes or refreshes. It stores data with no expiration date. The local storage stores strings. I learned it has two methods in the book but an extra reading about it on MDN showed another method. They are: the getItem(), setItem(), and removeItem() methods.

Local Storage Methods

  1. The getItem() method reads the data from the local storage. It takes one argument that is referenced as the key to search for and open the content/value.
  2. The setItem() method saves data to the local storage. It take two arguments- the key and the value.
  3. The removeItem() method removes the data from the local storage.

All data saved in the local storage is saved in the browser. The local storage saves a key/value pair and uses it to access the data in the background.

Questions
