Callbacks, High Order Functions and the Call Stack

Functions are more when they are reusable with more generic input or arguments. See an example of a simple function below. This function takes an array as an argument and loop through each value, multiplying each value by 2 and pushing each multiplication and assigning the new array or the array with changed values to a previously declared empty array. So that empty array will have the product of each element in the array you provided. 

            
              function copyArrayAndMultiplyBy2(array) {
                let output = [];
                for (let i = 0; i < array.length; i++) {
                  output.push(array[i] * 2);
                }
                return output;
              }
              const theArray = [1, 2, 3];
              let result = copyArrayAndMultiplyBy2(theArray);

Function callbacks allow more functionalities to be passed around in a high order functions. Callbacks are functions declared and called as arguments in another function. Callbacks make function more reusable. For example, another function can be created to give an instruction to the function to perform. 

            
              //The High Order Function- A generalized function that accepts instructions written in another function
              function copyArrayAndManipulate(array, instruction) {
                let output = [];
                for (let i = 0; i < array.length; i++) {
                  output.push(instruction(array[i]));
                }
                return output;
              }

              //The callback. It will be passed as an argument to the function
              function multiplyBy2(input){
                return input * 2;
              }
              let result= copyArrayAndManipulate([1,2,3], multiplyBy2);)
