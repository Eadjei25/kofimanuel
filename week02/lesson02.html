Testing and Debugging

Errors and bugs will always be there.

Kinds of Bugs

  1. Errors
    2. They are caused when something goes wrong
  2. Exceptions
    3. An exception is an error that produces a return value that can then be used by the program to deal with the error.
  3. Warnings
    4. A warning can occur if there’s an error in the code that isn’t enough to cause the program to crash.
