The Window Object

The window object is a global object in JavaScript. Because it is global, any global variables or functions can be used as properties or methods. The window object contains the webpage- the document object.

Browser Object Model

BOM for short is a collection of properties and method that contain information about the computer and browser screen. The properties and methods are made available through the window object. If you don't know the name of the global object, you can also refer to it using the keyword this in the global scope. The following code provides a quick way of assigning the variable global to the global object: 

            
                // from within the global scope
                const global = this;
