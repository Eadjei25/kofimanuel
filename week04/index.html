Forms

A form element contains form controls such as input fields, select menus and buttons.

Accessing Forms

Any of the DOM properties document.forms, document.getElementsByTagName() and document.getElementsByClassName() return an array of forms. The index notation is used to access the prefered form. Example: 

const form = document.getElementsByTagname('form')[0];

Forms can also be accessed uing their name. Eg: 

const form = document.forms.search;
or using the bracket notation: 
const form = document.forms['search'];

Accessing form controls or element in a form can be done using the elements property of the form object that returns an array of all the element in a form. Form controls can also be accessed using their name attribute value as if it was a property. This is done sing the dot or brackets notation
